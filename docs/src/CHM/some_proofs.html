명제 3.1 힐베르트 공간 \(\mathscr{H}\) 에서 정의된 연산자 \(M,\, N\) 에 대해 다음 \([M,\,N]=0\) 이면 \(e^{M+N} =e^M e^N\) 이다.


(증명).

MathExchange 의 질문과 답변 가운데 Robert Lewis 의 증명을 요약한다.

행렬에 대한 다음 상미분 방정식과 초기조건이 주어져 있다고 하자.

\[ \dfrac{dX}{dt}=(M+N)X \qquad \text{with }X(0)=I \tag{3.1}\]

이 미분방정식에 대한 유일해는 다음과 같다.

\[ X(t)=e^{(𝑀+𝑁)𝑡} \tag{3.2}\]

이다. 이제

\[ Y(t)=e^{Mt}e^{Nt} \tag{3.3}\]

라고 하자. 그렇다면 \[ \dfrac{dY}{dt}=\dfrac{d(e^{𝑀𝑡})}{dt} e^{Nt} + e^{Mt}\dfrac{d(e^{Nt})}{dt}= Me^{Mt}e^{Nt} + e^{Mt}Ne^{Nt} \tag{3.4}\]

이다. \([M,\,N]= MN-NM=0\) 이므로 \([e^{Mt},\, N]=0\) 이며 따라서

\[ \dfrac{dY}{dt} = Me^{Mt} e^{Nt} + Ne^{Mt}e^{Nt} = (M+N)Y \]

이고

\[ Y(0)=I \]

이다. \(X(t)\) 와 \(Y(t)\) 가 같은 미분방정식과 같은 초기조건을 가지므로 \(X(t)=Y(t)\) 이며, 따라서

\[ e^{(M+N)t} = e^{Mt} e^{Nt} \]

이다. \(t=1\) 에서 성립하며 이는 주어진 식이다. \(\square\)


명제 3.2 (Baker-Hausdorff Lemma) \(\lambda\) 가 실수일 때 다음이 성립한다. \[ \begin{aligned} e^{iG\lambda}Ae^{-iG\lambda} &= A + i\lambda [G,\,A] + \dfrac{(i\lambda)^2}{2!} [G,\,[G,\,A]] +\cdots \\ &\qquad + \dfrac{(i\lambda)^n}{n!} \underbrace{[G,\,[G,\,[G,\cdots [G}_{n\text{ times}},\,A]]]] + \cdots \end{aligned} \]


(증명).

MathExchange 의 질문과 답변 가운데 Robert Lewis 의 증명을 요약한다.

\(A,\,B\) 가 힐베르트 공간 \(\mathscr{H}\) 에서 정의된 연산자라고 하자. 에르미트여부는 상관 없다. 선형연산자에 대한 상미분 방정식

\[ \dfrac{dX}{d\lambda} = [B,\,X] \tag{3.5}\]

가 초기조건 \(X(0)=A\) 를 만족한다고 하자. 그렇다면

\[ X(\lambda) = e^{\lambda B}Ae^{-\lambda B} \tag{3.6}\]

는 이 미분방정식의 해라는 것을 보일 수 있다.

\[ \begin{aligned} \dfrac{dX}{d\lambda} = Be^{\lambda B} A e^{-\lambda B} - e^{\lambda B} A e^{-\lambda B}B = [B,\,X] \end{aligned} \tag{3.7}\]

이며 \(X(0)=A\) 이기 때문이다.

임의의 \(B\in \mathcal{L}(\mathscr{H})\) 에 대해 \(\phi_B:\mathcal{L}(\mathscr{H}) \to \mathcal{L}(\mathscr{H})\) 를 다음과 같이 정의하자.

\[ \phi_B(A) = [B,\, A] \tag{3.8}\]

연산자 노름 \(\| \cdot \|\) 에 대해

\[ \begin{aligned} \|\phi_B(A)\| &= \|[B,\,A]\| = \|BA-AB\| \\ & \le \|BA-AB\| \le \|B\|\|A\| + \|A\|\|B\| = 2\|A\|\|B\| \end{aligned} \]

가 성립한다. 즉 \(\|\phi_B\| \le 2\|B\|\) 이다. 이는 \(e^{\lambda B}\) 가 정의된다면 \(e^{\lambda \phi_B}\) 도 정의된다는 의미이다.

또한,

\[ \begin{aligned} \phi_B^2(A) &= \phi_B([B,\,A]) = [B,\,[B,\,A]], \\ \phi_B^3(A) &= [B,\, [B,\,[B,\,A]]] \end{aligned} \]

이고 다음을 쉽게 보일 수 있다.

\[ {\phi_B}^n (A) = \underbrace{[B,\,[B,\,[B,\cdots [B}_{n\text{ times}},\,A]]]]. \]

이제 \(Y(\lambda)\) 를 정해진 \(A,\,B\) 에 대해 다음과 같이 정의하자.

\[ Y(\lambda) = A+\lambda [B,\,A] + \dfrac{\lambda^2}{2!}[B,\,[B,\,A]] + \cdots + \dfrac{\lambda^n}{n!}\underbrace{[B,\,[B,\,[B,\cdots [B}_{n\text{ times}},\,A]]]] + \cdots \]

\(\phi_B\) 를 이용하여

\[ Y(\lambda) = \sum_{n=0}^\infty \dfrac{\lambda^n}{n!} {\phi_B}^n (A) = e^{\lambda \phi_B(A)} \]

임을 보일 수 있다. 이 \(Y(\lambda)\) 에 대해

\[ \dfrac{dY}{d\lambda} = \phi_B(e^{\lambda \phi_B(A)}) = [B,\, Y] \]

이며 \(Y(0) = A\) 임을 보일 수 있다. 즉 \(Y(\lambda) = X(\lambda)\) 이다. \(\square\)
